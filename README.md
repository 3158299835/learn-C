# C语言学习

我爱学习

学习C语言可以练习的一些题目

- **输入两个数，输出商和余数**
    
    ```c
    #include "stdio.h"
    
    int main()
    {
        int a = 0;
        int b = 0;
        //输入整数a和b 其中a>0  b<10,000
        scanf("%d %d", &a, &b);
        //计算输出
        printf("%d %d",a/b,a%b);
        return 0;
    }
    ```
    
    - **printf打印小飞机**
        
